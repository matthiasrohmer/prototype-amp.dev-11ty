가이드를 마쳤으며 HTML 문서를 AMP 문서로 변환하는 데 성공했습니다.

이 가이드에서 배운 내용은 다음과 같습니다.

- AMP 페이지와 기존 HTML 페이지 간 차이점 이해
- AMP 사이트 기초 익히기
- 기존 HTML 페이지를 AMP HTML로 변환
- AMP 페이지 유효성 검사
- AMP 콘텐츠 연결
- 검색엔진을 위한 구조화된 데이터 추가
